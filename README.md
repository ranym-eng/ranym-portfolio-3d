# 👩‍💻 Ranym Mejri — 3D Portfolio

This project is my personal portfolio as a Cloud Computing engineering student at ESPRIT.
It combines frontend development, 3D graphics, animations, audio interactions, and responsive design to create a more engaging experience than a classic portfolio website.

The goal is to showcase not only my projects, but also my ability to build modern, interactive, and well-structured digital experiences.

## ✨ Features

* Interactive 3D room
* Smooth animations
* Project showcase
* About and contact sections
* English / French language toggle
* Light / dark theme
* Audio and sound effects
* Responsive design

## 🛠️ Tech Stack

* Three.js
* Vite
* JavaScript
* SCSS
* GSAP
* Howler.js
* GLSL

## Google Analytics

Create a `.env` file from `.env.example` and replace `G-XXXXXXXXXX` with your Google Analytics measurement ID:

```env
VITE_GA_MEASUREMENT_ID=G-XXXXXXXXXX
```
